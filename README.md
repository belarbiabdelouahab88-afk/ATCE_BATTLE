# ATCE_BATTLE
L3 SPI EA projet ATCE avec jmf

un groupe de 5 étudiants en licence EEA à l'université UMLP travaillent sur le projet d'ATCE, ce projet consiste à concevoir une carte comprenant un microcontrôleur Atmega32U4 chargé d’asservir la fréquence du quartz qui le cadence sur un signal stable fourni par un récepteur GPS.
Ce depot contient tous les fichiers de la partie programmation du microcontroleur.
